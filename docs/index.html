Why mdcat

Everything you need, nothing you don't

Built on raw ANSI escape codes — no native deps, no Electron, no browser.

🎨
One Dark palette
Colour-coded headings, amber inline code, syntax-highlighted code blocks via highlight.js.
🔍
Incremental search
Press / to search. Matches highlight inline as you type. n / N to cycle.
🖱️
Mouse wheel scrolling
Full SGR mouse support. Scroll with your trackpad or mouse wheel — no key combos needed.
🔗
Clickable links
OSC 8 hyperlinks in iTerm2, Kitty, WezTerm, and foot. Click to open in your browser.
📊
Full GFM support
Tables with alignment, task lists, strikethrough, blockquotes, nested lists — all rendered.
⚡
Zero config
No dotfiles, no config, no setup. npx mdcat file.md and you're done.

Installation

Up and running in seconds

Three ways to install — or skip installation entirely with npx.

# No install — runs directly from npm registry
$ npx mdcat README.md
# Install once, use everywhere
$ npm install -g mdcat
$ mdcat README.md
# macOS via Homebrew tap
$ brew install frankchan/tap/mdcat

Requirements: Node.js 18 or later. Works on macOS, Linux, and Windows (WSL).
